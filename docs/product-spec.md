# MyToDoBar MVP Product Spec

## Goal

백엔드 개발 취업을 준비하며 떠오르는 오늘의 할 일과 기술 부채를 잊기 전에 메뉴바에서 즉시 기록한다. 과거 기록은 완료 여부와 함께 날짜별로 조용히 보존한다.

## MVP

- 메뉴바 아이콘을 누르면 오늘 TODO 팝오버가 열린다.
- 입력창에 자동으로 포커스된다.
- 텍스트 입력 후 Enter로 오늘 TODO가 저장된다.
- 체크박스로 완료하며 완료 항목은 당일 흐리게 남는다.
- 날짜가 바뀌면 이전 항목은 메뉴바에서 사라지고 원래 날짜의 기록으로 보존된다.
- 미완료 항목은 다음 날로 이동하거나 다시 알리지 않는다.
- 전체 보기는 Apple Calendar 느낌의 월간 달력과 날짜별 TODO 건수를 제공한다.
- 달력 상단에서 제목과 마감일시가 있는 예정 일정을 추가·수정·삭제하고 가까운 순으로 조회한다.
- 마감시간이 지난 일정은 예정 목록에서 자동으로 숨기되 로컬 데이터에는 보존한다.
- 날짜 선택 시 완료·미완료 목록을 함께 보여준다.
- 전체 보기에서 완료·미완료 TODO를 상태별로 필터링한다.
- 날짜별로 제목을 지정한 메모를 여러 개 작성하고 자동 저장한다.
- 선택한 메모를 설정한 GitHub 저장소와 브랜치의 메모별 선택 경로에 Markdown으로 게시한다.
- 동일 메모는 기존 원격 파일을 업데이트하고, 원격에서 변경된 파일은 덮어쓰지 않는다.
- GitHub 토큰은 macOS Keychain에 저장한다.
- 데이터는 Mac에 로컬 저장하고 재실행 후에도 유지한다.
- 로그인 시 자동 실행은 기본 OFF이며 설정에서 켤 수 있다.

## Non-goals

- 태그 및 카테고리
- iPhone 앱과 iCloud 동기화
- 일정 알림과 반복 일정
- TODO 마감시간과 자동 이월
- 오늘의 메모
- 반복 TODO와 우선순위

## Acceptance Criteria

1. 메뉴바 팝오버를 열고 텍스트 입력과 Enter만으로 TODO를 만든다.
2. 체크박스 한 번으로 완료 상태를 변경한다.
3. 완료 항목은 당일 흐리게 표시된다.
4. 과거 완료·미완료 항목은 원래 날짜에서 조회된다.
5. 미완료 항목은 자동 이월되거나 알림을 만들지 않는다.
6. 앱 재실행 후 데이터가 유지된다.
7. 자동 실행 설정의 초기값은 OFF다.
8. 전체 보기에서 전체·미완료·완료 TODO를 구분해서 조회한다.
9. 선택한 날짜의 여러 메모와 각 제목이 재실행 후에도 유지된다.
10. 동일 제목의 서로 다른 메모는 번호 접미사가 붙은 별도 Markdown 파일로 게시된다.
11. 게시 후 제목이 변경되면 원격 파일명을 변경하고, 앱에서 메모를 삭제해도 원격 파일은 보존된다.
12. 달력 상단에서 제목·날짜·시간이 있는 예정 일정을 추가하고 가까운 마감 순으로 조회한다.
13. 예정 일정은 수정·삭제할 수 있고, 마감시간이 지나면 목록에서 자동으로 숨겨진다.
